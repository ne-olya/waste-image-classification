# Проверка партии разметки

- [ ] Нет дубликатов одного объекта между train/val/test.
- [ ] Опасные отходы не попали в бытовые recyclable-классы.
- [ ] Проверены примеры `paper` против `cardboard`.
- [ ] Проверены примеры `glass` против прозрачного `plastic`.
- [ ] Неоднозначные multi-object фотографии исключены или помечены.
- [ ] Для каждого класса просмотрена случайная выборка не менее 50 изображений.
